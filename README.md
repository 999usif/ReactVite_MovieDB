# ReactVite project using themoviedb API
Features:
- See top trending movies
- Search for movies
- Favorite movies
- Store favorites locally using React Context

# Usage
Prerequisite: API key from themoviedb
1. Clone the repo
2. npm install
3. insert API_KEY in `src/services/api.ts` 
3. npm run dev
